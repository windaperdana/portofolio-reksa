## Getting Started

1. Clone or download the repository
2. Open `index.html` in your web browser
3. Customize the content to match your information

## Customization

### Personal Information
Edit the following sections in `index.html`:

- **Hero Section**: Update name, title, and description
- **About Section**: Modify personal information and statistics
- **Skills Section**: Update skills and technologies
- **Projects Section**: Replace with your own projects
- **Contact Section**: Update contact information

### Colors and Styling
Modify the CSS custom properties in `css/style.css`:

```css
:root {
    --primary-color: #667eea;
    --secondary-color: #764ba2;
    --accent-color: #f093fb;
    /* ... other variables */
}
```

### Images
Replace the placeholder images with your own:

- Profile image in hero section
- About section image
- Project screenshots
